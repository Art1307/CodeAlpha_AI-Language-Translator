## 📌 Project Overview

The **CodeAlpha AI Language Translator** is an AI-powered web application that enables users to translate text between multiple languages through an intuitive interface.

The application integrates Google Translation services to deliver fast and accurate translations while demonstrating the practical use of Artificial Intelligence and Natural Language Processing (NLP) in modern software development.

---

## ✨ Features

* 🌐 Translate text between 100+ languages
* 📝 User-friendly interface built with Streamlit
* 🔤 Select source and target languages
* 🤖 AI-powered translation
* 📋 Copy translated text with one click
* 🔊 Text-to-Speech support
* ⚡ Fast and lightweight
* 📱 Responsive web interface

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Art1307/CodeAlpha_AI-Language-Translator.git
```

### 2. Open the project

```bash
cd CodeAlpha_AI-Language-Translator
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```

---

## 📖 How It Works

1. Enter the text you want to translate.
2. Select the source language.
3. Select the target language.
4. Click the **Translate** button.
5. View the translated output.
6. Copy the translated text or listen to it using the Text-to-Speech feature.
